# FPBook Design System

> Manual de identidad visual del libro **CoppeliaSim para Formación Profesional**.

Este documento define el estilo gráfico, editorial y didáctico que se utilizará en todo el libro.

Su objetivo es garantizar que todas las páginas mantengan una apariencia uniforme, profesional y fácilmente reconocible.

## Contenido

- Colores corporativos.
- Tipografía.
- Iconografía.
- Figuras.
- Diagramas.
- Componentes.
- Plantillas.
- Reglas de estilo.
